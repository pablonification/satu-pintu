## Catatan untuk Tim UI/UX

### Prioritas Screen (untuk MVP)

| Priority | Screen | Notes |
|----------|--------|-------|
| P0 | Landing Page + Tracking Input | Simple, clean |
| P0 | Ticket Status Page | Show timeline |
| P0 | Dinas Login | Simple auth |
| P0 | Dinas Dashboard | List + filter |
| P0 | Ticket Detail (Dinas) | Update status |
| P1 | Error Pages (404, ticket not found) | User friendly |
| P2 | Admin Dashboard | Post-MVP |

### Design Guidelines

1. **Color Palette Suggestion**
   - Primary: Biru (trust, government)
   - Success: Hijau
   - Warning: Kuning/Orange
   - Error: Merah
   - Neutral: Abu-abu

2. **Typography**
   - Clean, readable font (Inter, Poppins)
   - Good contrast for accessibility

3. **Mobile First**
   - Warga akan akses via HP
   - Dashboard dinas mungkin via desktop

4. **Accessibility**
   - Besar font cukup
   - Kontras warna bagus
   - Icon + text (tidak icon saja)

5. **Branding**
   - Logo SatuPintu
   - Bisa include logo Kota Bandung
   - Government look-and-feel tapi modern
